使用python解释器

Posted on 2018-05-22

在看SSD的代码，总感觉看代码时，都是去百度函数，百度包，然后看完这个代码，具体还是不会用。于是重操python。参考从零开始的python世界的闯荡系列。

版本管理

现在用的pyenv进行python管理，具体教程可参见Python版本管理；服务器上使用的anaconda，可参见之前写的anaconda总结。进行版本管理的好处呢，可以隔离环境和多版本使用，pyenv使用时，还是需要管理员权限安装一些依赖；而anaconda不需要，你可以完全在你的用户目录下使用。

python解释器

直接键入python就可以调用python解释器，（解释器？），退出时，windows上按快捷键Control-Z，ubuntu上按Control-D，或者可以输入退出命令：quit()。

编码方式

默认使用UTF-8编码方式，若要使用其他编码，可以在代码的前面指定：# -*- coding: encoding -*-，encoding写你想指定的方式。另外需要在一开始加入#!/usr/bin/env python3。关于python中 #!/usr/bin/env python 与 #!/usr/bin/python的区别。
另外，我现在没有使用像pychrm这样的编辑器，用的简单的nodepad++；
