ubuntu安装啥啥啥

Posted on 2018-03-17

今天又想重新开始了，所以弄了个ubuntu16的系统盘，重装了下，安装步骤啥的呢就安装小哥博客里面的哦！

Chrome：
下载：wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
安装：sudo apt-get -f install
sudo dpkg -i google-chrome-stable_current_amd64.deb
参见Ubuntu 16.04安装Chrome浏览器

按照小哥博客装依赖，包。
sudo apt-get install build-essential
sudo apt-get install git
sudo apt-get install vim

安装pyenv：Python版本管理

python里面安装包：
h5py:sudo apt-get install libhdf5-dev 然后pip install。注意使用numpy==1.13.3
theano:sudo apt-get install libblas-dev liblapack-dev libatlas-base-dev gfortran
